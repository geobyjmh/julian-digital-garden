# Core Idea

Evidence alone does not create learning; insight comes from analysing and synthesising patterns.

# Learning Card Structure

1. Hypothesis
2. Observation
3. Learning & Insight
4. Decision & Action

# Affinity Mapping

A method for grouping qualitative observations into themes.

# Process

- Record observations
- Group similar evidence
- Identify themes
- Extract insights

# Key Insight

Patterns across experiments matter more than isolated observations.
