# Core Idea

Not all hypotheses are equally important. The most dangerous assumptions should be tested first.

# Prioritisation Framework

Hypotheses are ranked using:

- Criticality
- Existing evidence

# High-Risk Assumptions

The highest priority assumptions are:

- Critical to success
- Supported by little or no evidence

# Four Risk Categories

- Desirability
- Feasibility
- Viability
- Adaptability

# Key Insight

Testing low-risk assumptions first creates false progress.
