# Core Idea

In innovation work, progress is not measured by activity or output, but by reduced uncertainty.

# Traditional Metrics

- Features built
- Tasks completed
- Velocity

These do not prove the business idea works.

# Real Progress

- Better evidence
- Higher confidence
- Reduced risk
- Validated assumptions

# Innovation Scorecard

Measures progress across:

- Desirability
- Feasibility
- Viability
- Adaptability

# Key Insight

Learning faster is more valuable than building faster.
