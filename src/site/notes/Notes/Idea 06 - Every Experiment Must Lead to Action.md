# Core Idea

The purpose of testing is not running experiments — it is making decisions.

# Four Possible Actions

1. Test a new hypothesis
2. Continue testing the same hypothesis
3. Pivot the business model or value proposition
4. Kill the idea

# Confidence Depends On

- Strength of evidence
- Number of experiments
- Reproducibility
- Behavioural validation

# Important Principle

Weak evidence requires more testing before major decisions are made.

# Key Insight

A business testing process without action becomes research theatre.
