**Title:** The Business Model Canvas as a Shared Language

**Core Idea:**  
The [[Notes/Business Model Canvas\|Business Model Canvas]] provides a visual framework for describing how an organization creates, delivers, and captures value.

**Nine Building Blocks:**

- Customer Segments
    
- Value Propositions
    
- Channels
    
- Customer Relationships
    
- Revenue Streams
    
- Key Resources
    
- Key Activities
    
- Key Partnerships
    
- Cost Structure
    

**Key Insight:**  
The canvas is not a checklist; it is a system where all components influence one another.
